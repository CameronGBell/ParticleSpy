Welcome to ParticleSpy's documentation!
=======================================

ParticleSpy is a python package which eases segmentation and analysis of nanoparticles from electron microscopy data. 
It extends the functionality of the `Hyperspy <http://hyperspy.org/>`_ package in order to easily segment and analyse regions of Hyperspy signal objects.
